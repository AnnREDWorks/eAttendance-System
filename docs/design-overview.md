# eAttendance System Design Overview

## Purpose
Track attendance of employees/students and visualize trends.

## Planned Features

### Power Apps
- Login screen
- Check-in / Check-out buttons
- View attendance history
- Filter by date, employee, department

### Power Automate
- Trigger on check-in/check-out
- Save data to Dataverse / SharePoint / Excel
- Optional: Send confirmation email

### Power BI
- Attendance trends
- Department-wise reports
- Highlight late arrivals or absences
