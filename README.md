# eAttendance System

## Description
An employee/student attendance tracking system built with Microsoft Power Platform (Power Apps, Power Automate, Power BI) for portfolio purposes.

## Tech Stack
- Power Apps (Canvas app for UI)
- Power Automate (flows for attendance logging)
- Power BI (dashboard for visualization)
- Dataverse / SharePoint / Excel (backend storage)

## Features
- Check-in / Check-out attendance
- View personal or departmental attendance history
- Attendance trends and analytics

## Setup / Usage
1. Import `powerapps/eAttendance.msapp` into Power Apps.
2. Import Power Automate flows from `powerautomate/`.
3. Open Power BI dashboard from `powerbi/attendanceDashboard.pbix`.
